## Tips for Taking Screenshots

1. **Use consistent window size** (e.g., 1280x800)
2. **Clean up test data** before taking final screenshots
3. **Use example class names** like "Period 1 - Math", "Art Class"
4. **Add sample images** to levels for realistic appearance
5. **Consider using a screenshot tool** with annotation features

## Image Formats

- **PNG** recommended for UI screenshots (lossless, good for text)
- **JPG** acceptable for photos if file size is a concern
- **Keep file sizes reasonable** (<500KB per image when possible)
